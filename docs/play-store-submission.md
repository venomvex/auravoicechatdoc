## Step 1: Google Play Console Setup

### 1.1 Create Developer Account

1. Go to [play.google.com/console](https://play.google.com/console)
2. Pay $25 registration fee
3. Complete identity verification
4. Accept Developer Distribution Agreement

### 1.2 Create New App

1. Click "Create app"
2. Enter app details:
   - **App name**: Aura Voice Chat
   - **Default language**: English (US)
   - **App or game**: App
   - **Free or paid**: Free
3. Accept declarations
4. Click "Create app"

## Step 6: App Signing

### 6.1 Google Play App Signing (Recommended)

1. Let Google manage your app signing key
2. Upload your upload key
3. Benefits:
   - Key recovery if lost
   - Better security
   - Required for AAB

### 6.2 Generate Upload Key

```bash
# Generate upload key
keytool -genkey -v -keystore upload-keystore.jks \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias upload-key

# Export certificate
keytool -export -rfc -keystore upload-keystore.jks \
  -alias upload-key -file upload_certificate.pem
```

### 6.3 Configure in build.gradle

```groovy
android {
    signingConfigs {
        release {
            storeFile file("upload-keystore.jks")
            storePassword "your-store-password"
            keyAlias "upload-key"
            keyPassword "your-key-password"
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
        }
    }
}
```

## Common Rejection Reasons & Solutions

### 1. Privacy Policy Issues

**Problem**: Missing or incomplete privacy policy
**Solution**: 
- Host privacy policy on your website
- Include all required sections
- Make URL publicly accessible

### 2. Metadata Policy Violation

**Problem**: Misleading description or graphics
**Solution**:
- Accurate screenshots
- No fake reviews promised
- No competitor mentions

### 3. User Data Policy

**Problem**: Data safety form incomplete
**Solution**:
- Complete all sections
- Be transparent about data usage
- Provide data deletion option

### 4. Payments Policy

**Problem**: Virtual currency issues
**Solution**:
- Use Google Play Billing for in-app purchases
- Clear pricing before purchase
- Refund policy available

### 5. Restricted Content

**Problem**: Adult or gambling content
**Solution**:
- Age gate if necessary
- No gambling with real money
- Content moderation active

### 6. Broken Functionality

**Problem**: App crashes or features don't work
**Solution**:
- Thorough testing before submission
- Provide test credentials
- Fix critical bugs
