## Asset Guidelines

See [Logo Specification](../design/logo.md) for logo usage guidelines.

### File Naming Convention
- Use lowercase with underscores
- Include size suffix where applicable (e.g., `logo_48.png`, `logo_96.png`)
- Prefix by category (e.g., `medal_`, `frame_`, `gift_`)

### Formats
- PNG for raster graphics (with transparency where needed)
- SVG for scalable graphics (optional)
- WebP for optimized in-app images
