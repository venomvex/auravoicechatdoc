# Economy Balance

Master document for Aura Voice Chat economy balancing. All coin values, rewards, and cash amounts are calibrated for a stable, sustainable economy.

## Core Economy Principles

### Currency Foundation

| Currency | Type | Source | Sink |
|----------|------|--------|------|
| Coins | Primary | Purchase, rewards, earnings | Gifts, store, games |
| Diamonds | Secondary | Receiving gifts | Convert to coins (30%) |
| USD | Real money | Earnings, reselling | Withdrawals |

### Base Exchange Rates

| Conversion | Rate |
|------------|------|
| $1 USD → Coins | 100,000 coins |
| 1 Diamond → Coins | 0.30 coins |
| Coins → $1 USD (Earning) | 200,000 coins = $1 |

**Economy Ratio:** Users get back 0.5% of coins spent as cash earnings (anti-inflation)
