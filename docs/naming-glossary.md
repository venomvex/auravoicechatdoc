# Global Naming Glossary (Canonical Terms)

Use these exact terms everywhere (UI, docs, telemetry, CMS). Do NOT mix legacy names.

---

## Currency & Progress

| Term | Description |
|------|-------------|
| Star Coins | Main coin currency |
| Nova XP | Experience points |
| VIP Score | AuraPass tier progress |
| Recharge Coins | Cumulative monthly Star Coins (internal label only, display as "Monthly Recharge") |
