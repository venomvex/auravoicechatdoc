# Custom Animations & Effects

Comprehensive documentation for all custom animations, entry effects, gift animations, and visual effects in Aura Voice Chat.

## Overview

Aura Voice Chat features unique custom animations for various interactions including room entries, gift sending, profile effects, and UI elements. All animations are original designs.

## Animation Technical Specs

### Format Standards

| Type | Format | Max Size | Frame Rate |
|------|--------|----------|------------|
| Entry | Lottie JSON | 500KB | 60fps |
| Gift (small) | Lottie JSON | 100KB | 30fps |
| Gift (full-screen) | Lottie JSON | 2MB | 60fps |
| Frame | Lottie/GIF | 200KB | 30fps |
| Seat | Lottie JSON | 150KB | 30fps |

### Performance Guidelines

- Max 3 full-screen animations queued
- Auto-quality reduction on low-end devices
- Skip option after 2 seconds
- Reduce Motion setting disables animations

## Firebase Storage

### Asset Structure

```
/animations
  /vehicles
    /{vehicleId}.json
    /{vehicleId}_preview.png
  /gifts
    /{giftId}.json
    /{giftId}_preview.png
  /frames
    /{frameId}.json
    /{frameId}_static.png
  /themes
    /{themeId}/
      - frame.json
      - vehicle.json
      - seat.json
      - mic.json
      - bubble.json
```

### CDN Configuration

- Global CDN for asset delivery
- Edge caching for popular animations
- Lazy loading for non-immediate assets
- Fallback to static images if animation fails
